# Sokoban

Program to resolve the sokoban problem.

Initial state:

	 _________________
	|     |     |     |
	|     |     |     |
	|_____|_____|_____|
	|     |     |     |
	|     |  B  |  B  |
	|_____|_____|_____|
	|     |     |     |
	|  A  |  G  |  G  |
	|_____|_____|_____|

A: Position of the android (robot)

B: Position of a box

G: Position where the box must end.

Tested in macOS High Sierra with Ciao Prolog.
